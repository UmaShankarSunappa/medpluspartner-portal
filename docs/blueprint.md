# **App Name**: MedPlus Partner Portal

## Core Features:

- Pre-Login Experience & Authentication - Public Landing Page: Includes Header (Logo, Sign In, Sign Up), Navigation (About Us, News & Updates, FAQs, Resources & Training, Contact Us), Hero Banner (Become a Partner, Existing Partner Login), and Informational Sections (Why Partner, Mission Statement, Leadership Message)
- Pre-Login Experience & Authentication - Authentication: Login (Email/Username, Password, Forgot Password) and Controlled Sign-Up (Activation via Admin for approved franchisees)
- Home (Dashboard): Welcome Banner, KPI Cards (Active Orders, Available Credit, Pending Returns, Open Tickets), and Multi-Store Selector (filters dashboard data per store)
- Account Profile: My Profile (Personal/Business Details), License Details (Store ID, GST, DL, FSSAI, etc.), Bank Account Details, and Edit / Save Functionality
- Orders - Order History: Date filters (15-day range), Table (Order ID, Date, Total, Status, Type), and Actions (View, Re-Order, Track Order)
- Orders - Provisional Indent: Auto-generated draft orders, Review & Edit items, and Confirm Indent
- Financial Management - Account Balance: Summary (Credit Limit, Outstanding, Credit Period) and Transaction Statement (90-day range, printable/exportable)
- Financial Management - Payments: Payment History (30-day default, searchable) and Create Payment (Form + Workflow)
- Returns: Return History (filterable, list of returns) and Return Detail Pop-up (products, quantities, pricing, status)
- Reports - Performance Reports (Dashboard): Filters (Store, Date Range) and Visual KPIs (Sales, Margins, Discounts, Orders, ABV, Bounce Products, SKU Replenishment, etc.)
- Reports - Monthly Reports: Franchisee View (Table of downloadable reports (Excel/PDF), Report Types, Action Required: Confirm Returns with dispatch date) and Admin View (Uploads, Bulk Distribution, Tracking Compliance)
- Invoices - Order Invoices (From Franchisor): Auto-uploaded PDFs, searchable & downloadable, and Multi-store filtering
- Invoices - Membership Agent Fee Invoices (To Franchisor): Auto-populated commission data, Digital acceptance â†’ PDF invoice generation, and Backend accounts dashboard for status updates
- Field Team Store Visit Reporting: Actors, Input Form, Process Flow, Outputs
- Complaints Management: Franchisee View, Backend View, Notifications

## Style Guidelines:

- Primary color: Light desaturated blue (#94BDBD), suggestive of reliability and health.
- Background color: Very light grey (#F0F0F0) for a clean, professional look.
- Accent color: Teal green (#63B7AF), provides contrast with the primary, but still blends in well.
- Headline font: 'Poppins', a geometric sans-serif font.
- Body font: 'PT Sans', a humanist sans-serif font; a good complement to Poppins if the content uses both headlines and body text extensively.
- Use clear, modern icons to represent different functions and categories within the portal.
- A clean and intuitive layout with a sidebar navigation for easy access to all features.
- Subtle transitions and animations to enhance user experience without being distracting.
